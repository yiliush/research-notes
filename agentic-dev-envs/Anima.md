# Anima

Anima is the design-to-code platform that pivoted hardest into the agentic-IDE era — adding an in-browser AI Playground (full app prototyping, built-in databases, one-click publishing), an MCP server installable in Cursor and Claude Code, a Frontier VS Code extension that maps generated code to existing components, and an IBM investment in February 2026 that validated its enterprise positioning.

## Origin
- Anima App founded 2017 by Avishay Cohen, Jonathan Cohen, and Michael Cohen. Tel Aviv / San Francisco.
- Earliest entrant in the Figma-to-code space; originally focused on pixel-perfect HTML/CSS export.
- 2023–2024: shifted from pure code export to AI-assisted generation; added React and Vue outputs.
- February 2026: IBM investment, validating enterprise positioning.

## Adoption
- One of the longest-running Figma plugins in the design-to-code category.
- IBM partnership extends enterprise reach.
- Anima MCP server self-installable in Cursor and Claude Code — riding the agentic-IDE wave rather than competing against it.

## Product
- Figma plugin — code inspection and export across React, Vue, plain HTML/CSS, Tailwind CSS, and email-ready HTML for Outlook-compatible newsletter templates.
- AI Playground (browser sandbox) — build and iterate on apps with natural-language prompts, image inputs, or URL cloning. Built-in databases detected and provisioned automatically. One-click publishing.
- Frontier (VS Code extension) — maps generated code onto existing components in the user's codebase, narrowing the "AI-generated code looks foreign" gap.
- Anima API — automates Figma-to-code and website-to-code for AI coding pipelines.
- Anima MCP server — self-installable in Cursor, Claude Code, and other MCP-aware agents using an Anima account login.

## Pricing
- Free — 5 AI chat messages/day in Playground, 5 Figma imports or website clones, 5 code generations via the plugin.
- Pro — $24/month (annual billing).
- Enterprise — from $500/month — SSO, MFA, enterprise SLAs, compliance.

## Differentiators
- Most explicit MCP-server posture in this sub-cluster — Anima's bet is to be the design-to-code tool that an agent calls, not a standalone app.
- Frontier (component-mapping into existing codebases) addresses the biggest weakness of design-to-code: generated code that doesn't match the project's component library.
- AI Playground with built-in databases is closer to Bolt/Lovable territory than to Builder.io/Locofy — Anima is the only design-to-code vendor that also tries to be a full prompt-to-app surface.
- IBM investment signals enterprise traction in a category where most competitors are mid-market.
- Risk: trying to be both Figma plugin AND prompt-to-app platform stretches the product across two competitive fronts.
