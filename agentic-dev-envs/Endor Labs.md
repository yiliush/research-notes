# Endor Labs

Endor Labs is the fastest-growing entrant in the developer security space — $93M Series B in April 2025, 225% YoY revenue growth, and a strategic GitHub Advanced Security integration that puts them directly inside the GitHub workflow. Their technical wedge is reachability analysis: instead of flagging every CVE in your dependency tree, they trace whether vulnerable code is actually called.

## Origin

Founded in 2021 by Varun Badhwar (CEO, ex-Prisma Cloud at Palo Alto Networks) and Dimitri Stiliadis. Strong founder pedigree — Badhwar previously sold RedLock to Palo Alto for $173M.

## Funding & Traction

- Total raised: $188M.
- Series B (April 2025): $93M.
- ARR (2025): ~$13.6M per Getlatka — modest in absolute terms but 225% YoY growth.
- GitHub Advanced Security integration partnership announced 2025.

## Product

Endor Labs unifies several security functions:

1. **SCA with reachability.** ~92% reduction in false positives by tracing whether vulnerable code paths are actually reachable from your application's entry points.
2. **SAST.** Static analysis for first-party code.
3. **Secrets scanning.**
4. **Container scanning.**
5. **CI/CD security.**
6. **AURI Platform.** Integrates security into AI-driven code workflows — Endor's answer to the agentic coding wave.

## Pricing

Sales-led; not publicly disclosed. Likely mid-five to six-figure annual contracts based on growth profile.

## Differentiators

- **Reachability analysis.** The 92% false-positive reduction is a real technical win — security teams hate alert noise.
- **Single platform.** SCA + SAST + secrets + containers without four separate vendors.
- **GitHub partnership.** Native integration is a powerful distribution advantage.
- **Veteran founder.** Badhwar has done this exit before.

## Risk factors

- Competes head-on with [[Snyk]] which has 20x the revenue and entrenched distribution.
- 225% growth on a $13.6M base is impressive but absolute scale is small.
- The "reachability" pitch only works if customers care about prioritization — many enterprise security teams measure on coverage instead.

## Notable customers

Not widely disclosed; growth and partnership signals suggest mid-market and enterprise.
