# Codev

Codev (co.dev) is a Next.js-specialist prompt-to-app builder. It is the
narrower, indie-hacker-leaning answer to v0 — same stack opinion, less
Vercel pull, more focus on shipping a single product start-to-finish.

## Origin

Codev positioned itself from launch as a Next.js-first AI app builder.
The product hypothesis: if you commit to one framework rather than
trying to support many, the agent gets better at that framework
specifically, and the user gets cleaner output. Same logic as v0, but
without Vercel's distribution advantage — Codev has to win on craft.

## Funding & Traction

- Specific funding rounds not publicly disclosed at venture-tracked
  granularity
- Operates as a small-team product with a free starter tier

## Product

Chat-driven app generation with full Next.js scaffolding: App Router,
server actions, Tailwind, typed environment, deployment story. The
"Plus" tier exposes private projects, code download, GitHub integration,
and custom domains — the standard indie hacker checklist.

**Generated stack**: Next.js (App Router), TypeScript, Tailwind. GitHub
sync is first-class. The user is expected to graduate from the platform
into their own repo at some point.

**Deployment**: Standard one-click deploy targets. Less locked-in than
v0-on-Vercel.

| Enterprise | custom | team onboarding, support, customisation |

The 25% off AI model usage on the Plus tier is unusual — most
competitors do not pass through model cost discounts to subscribers.

## Differentiators

- **Next.js purist.** Stack opinion is identical to v0; what differs is
  the lack of Vercel pull-through. The output is meant to live in your
  own GitHub and deploy wherever you like.
- **GitHub-first.** Code download and GitHub integration are core to
  the workflow, not afterthoughts. This is a signal the platform expects
  users to eventually leave (and is comfortable with that).
- **Free $3 starter credit** is a low-friction way to evaluate without
  a payment method on file.
- **Smaller scale** than v0 or Bolt — closer to a craft tool than a
  hyperscaling platform.
