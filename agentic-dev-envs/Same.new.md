# Same.new

Same.new is the website cloning tool in this cluster. Paste a URL, get
back production-ready code that looks like the original. It is the
narrowest product in the vibe coding category and the cheapest.

## Origin

Same.new launched as a single-purpose tool: clone any public website
into clean, editable code. The pitch is honest about what it is — a
starting point for a redesign or a learning exercise — not a full app
builder. The product is small enough that it competes on price and
speed rather than feature surface.

## Funding & Traction

- No public funding disclosure
- No public ARR disclosure
- Operates as a small focused tool, not a venture-scale platform

## Product

Paste a URL. The AI analyses the page (layout, components, styling) and
emits clean code that reproduces the design. Users can then customise
the clone, add their own backend, and ship.

**Generated stack**: Modern frontend (React/Next.js-flavoured output).
The output is meant to be a foundation, not a fully wired backend app.

**Deployment**: Download the code or deploy through the platform.

## Pricing (May 2026)

- Free tier with basic URL cloning
- $10/month for 2M tokens

At $10/mo, Same.new is among the cheapest paid plans in the cluster.

## Differentiators

- **URL as primary input.** Most other platforms take text prompts.
  Same.new takes a website. This is materially different for designers
  doing teardowns or agencies prototyping a "we could do better than
  this" pitch.
- **Single-purpose clarity.** The product knows what it is. It is not
  trying to be Lovable or Bolt.
- **Cheap.** $10/mo with 2M tokens is a meaningful allowance for a
  focused tool.
- **Ethical surface area.** Cloning competitor sites is a grey area
  legally and reputationally — the product's bet is that "inspired by"
  redesign work is a legitimate use case big enough to support a
  business.
