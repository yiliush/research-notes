# a0.dev

a0.dev is the mobile-first vibe coding platform. Y Combinator W25.
Generates native iOS and Android apps via React Native, with the deploy
target being the App Store and Google Play — not a web URL.

## Origin

Founded as a YC W25 company. The thesis: every other platform in this
category outputs web apps. Mobile is harder — the toolchain, the build
servers, the store submission — but it is where most consumers actually
spend time. a0.dev took the bet that "prompt to App Store" was a
distinct enough product to win mobile-first builders.

## Funding & Traction

- Y Combinator Winter 2025 batch
- No public ARR disclosure
- Available as a Google Play app for users who want to build apps from
  their phone

## Product

A chat-driven agent that generates full React Native apps, including
UI, backend logic, payments, and store submission artifacts. The
platform handles iOS builds, debugs with a "thinking mode," and pushes
to the App Store and Google Play in one flow.

**Generated stack**: React Native. Bundled payments. Growth analytics
hooks. The platform handles the build server and store submission
plumbing that normally requires an Apple Developer account dance.

**Deployment**: App Store + Google Play. Not just a web URL — the
deliverable is a native binary.

## Pricing (May 2026)

- Free tier available
- Paid: ~$16/mo billed annually, ~$20/mo billed monthly

Pricing is notably lower than the web-first competitors, possibly
because mobile output is hard to evaluate without the store dance, and
the platform absorbs the build infrastructure.

## Differentiators

- **Native mobile output**, not a PWA or web wrapper. React Native
  compiles to real iOS and Android binaries.
- **Store submission included.** Closing the loop on Apple Developer
  certificates and Google Play signing is the actual differentiator —
  this is where most "I built a mobile app with AI" projects die.
- **Mobile-as-input.** The Google Play app means a user can build an
  app from their phone. None of the major web competitors offer this.
- **Cheap relative to Bolt/Lovable**, which makes sense as a wedge into
  the mobile builder market.
