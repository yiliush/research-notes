# Snyk

Snyk is the largest specialty agent in the security cluster — a late-stage incumbent in developer security with $1.32B raised, $278M in 2024 revenue, and an aggressive 2025 pivot to position itself as the "AI Security Fabric" for agentic development. The Evo agentic orchestration system, launched October 2025, is Snyk's bet that AI-generated code will need AI-driven security at the same speed.

## Origin

Founded in 2015 by Guy Podjarny, Assaf Hefetz, and Danny Grander. Tel Aviv / London origins. Snyk pioneered "developer-first security" by embedding SCA into npm install flows and Git hooks. IPO has been on-and-off the table for years.

## Funding & Traction

- Total raised: $1.32B across 17 rounds from 77 investors.
- 2024 revenue: $278.4M (26.5% YoY growth — a deceleration from earlier years).
- Q2 2025 growth slowed to 12% YoY, prompting concern about IPO timing.
- Acquisitions: Probely (DAST, Nov 2024), Invariant Labs (AI security, June 2025).

## Product

Snyk's platform spans:

1. **Snyk Code (SAST).** Static analysis for first-party code.
2. **Snyk Open Source (SCA).** Vulnerability scanning for dependencies.
3. **Snyk Container.** Container image scanning.
4. **Snyk IaC.** Infrastructure-as-code policy.
5. **Snyk Agent Fix.** Autonomously generates and validates security fixes — sits in PRs.
6. **Evo by Snyk (Oct 2025 launch).** Agentic orchestration layer for securing AI-native apps and agents themselves.

## Pricing

- Free tier for individuals and small teams.
- Team: $25/dev/month (Snyk Code).
- Business: custom.
- Enterprise: custom — large F500 contracts.

## Differentiators

- **Established distribution.** Already in tens of thousands of organizations.
- **Multi-product breadth.** SAST + SCA + IaC + containers in one platform.
- **AI security investments.** Acquired Invariant Labs to position for agent-era security.
- **Snyk Agent Fix.** One of the few security tools that auto-generates PRs to fix, not just identify.

## Risk factors

- Growth deceleration (26% to 12%) raises questions for an IPO candidate.
- [[Endor Labs]] is growing 225% YoY and aggressively winning SCA deals.
- [[Apiiro]] and ASPM vendors compete for the same enterprise budgets.
- LLM-era security ([[DryRun Security]]) reframes SAST in ways Snyk has to catch up to.
